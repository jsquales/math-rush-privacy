Privacy Policy for Math Rush: Multiplication

Effective date: October 2025

Math Rush: Multiplication is designed for students and classrooms. We collect the minimum data needed to provide optional cloud features like Friends leaderboards, while allowing the app to work fully offline.

What we collect

No email, phone number, or precise location is collected. We do not show ads.

  • Pseudonymous identifier: A Firebase user ID (“friend code”) created by the app. No sign-up is required and no password/email is collected.
  • Nickname (optional): A name you choose in-app. This can be synced to the cloud so friends can recognize you.
  • Gameplay stats: Personal bests and session summaries (e.g., mode, difficulty, attempted, correct, average time per question, streak/duration, and end reason, with timestamps).
  • Friends list: Only the friend codes (Firebase UIDs) you add manually. There is no public search.

How we use data

  • Core app features: Run the game, show your personal bests, and compute fluency trends.
  • Optional cloud features: Sync your nickname and bests, and show Friends leaderboards when you add friend codes.

Where data is stored

  • On your device: Local progress and settings.
  • In the cloud (if enabled): Google Firebase (Authentication and Cloud Firestore) in order to store your pseudonymous user ID, nickname (if provided), gameplay stats, and the list of friend codes you chose to add.

Data sharing and selling

  • We do not sell your data.
  • We do not share data with advertisers.
  • We use Google Firebase as a service provider to operate authentication and the database. Data is encrypted in transit and stored by Google with encryption at rest.

Children & education use

Math Rush is suitable for school use and follows a minimal-data approach. If a student uses the Friends feature, they must manually exchange friend codes with people they know. Parents/guardians or schools can request deletion at any time (see below).

Your choices and controls

  • Use offline: You can play without using Friends or cloud sync.
  • Nickname: You can set or change your nickname within the app.
  • Friends: Add or remove friends at any time using their friend code (UID). There is no universal search.
  • Data deletion: You can request deletion of cloud data at any time (see next section).

Request data deletion

To delete your cloud data (Firebase user record, friends list, and stored bests/stats):

  1. Open the app and go to Friends → Your Friend Code to copy your code (UID), or include your chosen nickname and device/platform.
  2. Email phi1123apps@gmail.com with the subject “Math Rush – Delete My Data”.

We’ll confirm and delete associated cloud data. Local data on your device can be removed by uninstalling the app.

Security

  • Data is encrypted in transit and stored by Google with encryption at rest.
  • We implement Firestore security rules so each user can write/read only their own data, and can read limited “public bests” for friend codes they add.

No ads, no trackers

We do not include advertising SDKs or analytics/behavioral tracking SDKs. Crash reporting/analytics are not enabled.

Changes to this policy

We may update this policy as we add features (for example, classroom leaderboards). We will update the “Effective date” above and publish changes at the same URL.

Contact

If you have questions, or to request data deletion, contact: phi1123apps@gmail.com
